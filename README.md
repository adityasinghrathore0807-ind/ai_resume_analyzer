##  Overview

**Smart AI Resume Analyzer** is an intelligent web application that helps job seekers optimize their resumes for Applicant Tracking Systems (ATS).

Simply upload your resume, enter the target company, job title, and job description. The AI analyzes your resume and generates a detailed report containing:

-  ATS Score
-  Executive Summary
-  Strengths
-  Weaknesses
-  Missing Keywords
-  Formatting Review
-  Improvement Recommendations
-  Final Recruiter Verdict

The application also allows users to download a professional PDF report.

#  Installation

## Clone Repository

```bash
git clone https://github.com/adityasinghrathore0807-ind/ai_resume_analyzer.git
```

Move into the project directory

```bash
cd ai_resume_analyzer
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open your browser

```
http://localhost:5175
```

#  How It Works

### Step 1

Enter:

- Company Name
- Job Title
- Job Description

### Step 2

Upload your Resume (PDF)

### Step 3

The AI analyzes:

- Resume Content
- ATS Compatibility
- Skills
- Keywords
- Resume Structure
- Formatting

### Step 4

Receive:

- ATS Score
- Executive Summary
- Strengths
- Weaknesses
- Missing Keywords
- Formatting Review
- Improvement Suggestions
- Final Verdict

### Step 5

Download your professional Resume Analysis Report in PDF format.
